# Inter-Department-Sports-Dashboard
It is to track live scores, standings, and schedules for TCE's inter-department sports tournaments. Fully responsive, it features interactive filters, leaderboards, and easy score updates via JS data arrays—no backend required. Designed for TCE students and staff to follow the 2026 season events organized by the Physical Education department
TCE Inter Department Sports Dashboard 2026 is a lightweight web app built with HTML5, CSS3, and vanilla JavaScript to track live scores, department standings, and schedules for Thiagarajar College of Engineering's inter-department sports tournaments like cricket, basketball, football, and athletics. Fully responsive for mobile use at sports grounds, it features interactive filters, leaderboards, and easy score updates via JS data arrays—no backend required. Designed for TCE students and staff to follow the 2026 season events organized by the Physical Education department.
